Explicación para la presentación

En esta página usamos el sistema de rejilla de Bootstrap. Primero se coloca un container para centrar el contenido. Después se usa una row para crear una fila. Dentro de la fila se colocan columnas con clases como col-12, col-md-6 y col-lg-3.

Bootstrap divide cada fila en 12 columnas. Por eso, una columna con col-md-4 ocupa 4 partes de 12, y tres columnas de 4 pueden entrar en una misma fila. Los breakpoints permiten que la página se adapte a celular, tablet y computadora.
